# Security Policy

If you believe you've found a security vulnerability in Sequence Tool or any related service, please report it privately so we can investigate and fix it.

## Reporting a vulnerability

Please email:

- security@sequencetool.com

When reporting, include as much of the following as possible:

- A clear description of the issue and potential impact
- Steps to reproduce (proof-of-concept if available)
- Affected versions/builds (if known)
- Any relevant logs, crash reports, or screenshots

## Coordinated disclosure

Please do not open public GitHub issues for security vulnerabilities. Use the email address above so we can coordinate a fix and disclosure timeline.

## Acknowledgements

If we confirm a valid vulnerability report, we may acknowledge your contribution (with your permission) on our acknowledgements page:

- https://sequencetool.com/security/acknowledgements
